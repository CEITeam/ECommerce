### 📝 Description
A robust, full-stack E-commerce platform designed with an emphasis on **Scalability** and **Maintainability**. This project was developed as a collaborative effort within the **CEITeam** as a final course project during the **Civil Engineering Informatics Diploma at ITI (Information Technology Institute)**, demonstrating professional software engineering practices including **Clean Architecture**, **SOLID principles**, and **Design Patterns**.

---

### 🏗️ Architecture & Design
The system follows a decoupled architecture to ensure separation of concerns:
* **Core Layer:** Contains domain entities and business logic.
* **Application Layer:** Handles DTOs, Mapping, and Application Services.
* **Infrastructure Layer:** Manages data persistence, Entity Framework Core, and external integrations.
* **Web/Presentation Layer:** ASP.NET Core MVC/Web API for handling client requests.

---

### ✨ Key Features
* **Identity Management:** Secure authentication and authorization for users and admins.
* **Product Catalog:** Dynamic product management with categorization and search.
* **Shopping Cart & Orders:** Full checkout workflow from cart management to order persistence.
* **Database Management:** Efficient data handling using **Entity Framework Core** and **SQL Server**.
* **Clean UI:** Responsive interface designed for a seamless user experience.

### 💻 Tech Stack
* **Backend:** C#, ASP.NET Core, Entity Framework Core, LINQ.
* **Database:** Microsoft SQL Server.
* **Frontend:** HTML5, CSS3, JavaScript (Bootstrap).
* **Patterns:** Repository Pattern, Unit of Work, Dependency Injection.

### ⚙️ Getting Started
1.  Clone the repository.
2.  Update the connection string in `appsettings.json`.
3.  Run `Update-Database` in the Package Manager Console.
4.  Run the application via Visual Studio.
